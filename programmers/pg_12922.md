# x만큼 간격이 있는 n개의 숫자

https://programmers.co.kr/learn/courses/30/lessons/12922

```java
class Solution {
    public String solution(int n) {    
        String su = "수";
        String watermelon = "수박";
        int num = n/2;
        
        if(n == 1){
            return su;
        }else{
            if(n % 2 == 1){
                String re = watermelon.repeat(num);
                return re+su;
            }else{
                return watermelon.repeat(num);
            }
        }
    }
}
```
